# SpringBoot-Project-BLOG  
`SpringBoot` `JSP` `JPA` `MySQL` 를 사용하여 웹 게시판 구현

<br>

## :wrench: 사용 기술
```sh
Language : JDK 1.8, JSP 2.3
Framework : SpringBoot 2.7.8
Database : MySQL 5.7
ORM : JPA(Hibernate)
IDE : sts4 4.11.0
```

<br>

## :pushpin: 구현 기능
- 회원 기능
  * 회원가입
  * 카카오 회원가입
  * 로그인/로그아웃
  * 회원 정보 수정
- 게시판 기능
  * 게시글 상세보기
  * 게시글 작성
  * 게시글 수정
  * 게시글 삭제
  * 페이징
- 댓글 기능
  * 댓글 작성
  * 댓글 삭제
